# Интенсив №1. Ассемблер в задачах защиты информации

В репозитории размещены материалы по интенсиву №1 по дисциплине «Ассемблер в задачах защиты информации».

## Реализованные задания

1. XOR-шифрование строки фиксированным ключом.
2. Обнаружение переполнения буфера с использованием канарейки.
3. Проверка пароля с солью.
4. Построение цифровой подписи на основе RSA-подобной схемы.
5. Шифрование с использованием операций NOT и XOR.

## Среда разработки

Программы были реализованы на языке ассемблера MASM32 и тестировались в операционной системе Windows 7.

## Структура репозитория

- `asm/` — исходные коды программ на MASM32;
- `screenshots/` — скриншоты выполнения программ;
- `report/` — отчёт по интенсиву.

## Сборка и запуск

Исходные файлы открывались в MASM32 Editor и собирались стандартными средствами MASM32. После сборки запускались полученные `.exe`-файлы.
